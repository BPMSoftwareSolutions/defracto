# Projections

This folder holds published surfaces that have been promoted out of the conveyor.

The projection roots are now split by surface type:

- `visual-models/` for declared visual projection models
- `target-sketches/` for reference sketches and targets
- `ascii/` for ASCII projection outputs
- `markdown/` for markdown projection outputs
- `docs/` for document projections
- `slides/` for slide projections

Projection files are not the source of truth. They are outputs with lineage.
