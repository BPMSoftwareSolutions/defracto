# Evidence Inspector / Truth Reconciler Source Narrative

Artifact Key: `defracto.source.evidence-inspector.v1`

## Narrative

The Evidence Inspector / Truth Reconciler is the conveyor character responsible for **judgment**.

It exists so that the Defracto/NDD conveyor can keep each responsibility bounded, visible, receipt-backed, and unable to become a competing narrative.

## Owns

- receipt validation
- content mismatch disposition
- truth reconciliation
- pass/block judgment

## Does Not Own

- silent repair
- artifact invention
- worker output mutation
- shipping

## Doctrine

```text
The CLI opens.
The runner moves.
The workers transform.
The ledger remembers.
The inspector judges.
The shipper releases.
Telemetry reflects.
The conveyor lives.
```

## Completion Posture

This character is fully governed only when its story, canonical spec, SEJ contract, DTO contracts, projection manifest, generated surface, and receipts all align.
