# Lane Workers Source Narrative

Artifact Key: `defracto.source.lane-workers.v1`

## Narrative

The Lane Workers is the conveyor character responsible for **bounded transformation**.

It exists so that the Defracto/NDD conveyor can keep each responsibility bounded, visible, receipt-backed, and unable to become a competing narrative.

## Owns

- one lane input
- one lane output
- one lane receipt
- bounded transformation

## Does Not Own

- whole pipeline truth
- previous lane repair
- downstream release
- hidden fallback

## Doctrine

```text
The CLI opens.
The runner moves.
The workers transform.
The ledger remembers.
The inspector judges.
The shipper releases.
Telemetry reflects.
The conveyor lives.
```

## Completion Posture

This character is fully governed only when its story, canonical spec, SEJ contract, DTO contracts, projection manifest, generated surface, and receipts all align.
