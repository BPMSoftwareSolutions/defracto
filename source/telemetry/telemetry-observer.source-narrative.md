# Telemetry Observer Source Narrative

Artifact Key: `defracto.source.telemetry-observer.v1`

## Narrative

The Telemetry Observer is the conveyor character responsible for **reflection**.

It exists so that the Defracto/NDD conveyor can keep each responsibility bounded, visible, receipt-backed, and unable to become a competing narrative.

## Owns

- runtime observations
- feedback signals
- post-ship status
- source promotion candidates

## Does Not Own

- canonical truth before promotion
- release authority
- silent product changes

## Doctrine

```text
The CLI opens.
The runner moves.
The workers transform.
The ledger remembers.
The inspector judges.
The shipper releases.
Telemetry reflects.
The conveyor lives.
```

## Completion Posture

This character is fully governed only when its story, canonical spec, SEJ contract, DTO contracts, projection manifest, generated surface, and receipts all align.
