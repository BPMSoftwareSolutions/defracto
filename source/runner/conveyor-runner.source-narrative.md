# Conveyor Runner Source Narrative

Artifact Key: `defracto.source.conveyor-runner.v1`

## Narrative

The Conveyor Runner is the conveyor character responsible for **movement**.

It exists so that the Defracto/NDD conveyor can keep each responsibility bounded, visible, receipt-backed, and unable to become a competing narrative.

## Owns

- run envelope
- lane dispatch
- status ledger
- handoff sequencing
- sync and async progression

## Does Not Own

- lane content meaning
- truth judgment
- shipping authority
- worker response invention

## Doctrine

```text
The CLI opens.
The runner moves.
The workers transform.
The ledger remembers.
The inspector judges.
The shipper releases.
Telemetry reflects.
The conveyor lives.
```

## Completion Posture

This character is fully governed only when its story, canonical spec, SEJ contract, DTO contracts, projection manifest, generated surface, and receipts all align.
